# Тестовое задание Frontend middle в Mindbox

## Как запустить

1. **Клонируйте репозиторий:**

   ```
   git clone https://github.com/myr-irina/mindbox-test-task
   ```

2. **Установите зависимости:**

   ```
   npm install
   # или
   yarn
   ```

3. **Запустите проект:**

   ```
   npm start
   # или
   yarn start
   ```

4. **Откройте в браузере:**
   ```
   http://localhost:3000
   ```

## Технологии

- React
- TypeScript
- Ant Design
- CSS-модули
- Jest
